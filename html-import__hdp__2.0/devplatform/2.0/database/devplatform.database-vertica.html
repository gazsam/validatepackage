Managing a Vertica Preview Cluster from the Management Console

The Vertica Management Console offers the following features for database administrators and users:

  • Database administrators can use the Management Console to monitor cluster activity and performance, the Activity and Manage sections to view cluster and node specific performance, and configure the cluster through the Settings section.
  • Database users can use the Management Console's database designer to design optimized projections for schemas, and the EXPLAIN features to analyze the details of queries and projections.
Note: The Vertica Management Console is only available for Vertica Preview clusters. It is not available for Vertica Preview instances.

To access the Vertica Management Console, in Horizon, select Project, Database, Clusters, and select the cluster in the Cluster Name column. Select the Management Console link in the Overview tab.

Viewing Logs

Vertica event log information can be accessed via the ACTIVE_EVENTS table. Query history can be accessed via the QUERY_EVENTS table and user-issued query requests via the QUERY_REQUESTS table.
