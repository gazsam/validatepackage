HPE Helion 2.0 Development Platform: Using the Database Service

The database service provides lifecycle services for MySQL, Vertica Preview, Redis, or MongoDB database instances. Lifecycle services include provisioning, configuration, patching, backups, restores, and monitoring that can be administered from either a CLI interface, RESTful APIs or the Horizon dashboard.

These instances can then be utilized by your applications to provide standard datastore/back-end functionality.

Databases can be either single-instance or clustered. For production environments, clustered instances are recommended.

Interacting with the database at the command line is done with the Trove client. You can install the Trove client with the following command:

pip install python-troveclient
