Retrieving the Administrator Password for Windows instances

When a new Windows instance is created in HPE Helion OpenStack or Helion Public Cloud, a temporary administrator password is created. This topic demonstrates how to retrieve the administrator password for a new Windows instance.

This topic contains the following sections:

  • Retrieve the Windows instance password (Public Cloud)
  • Retrieve the Windows instance password (HPE Helion OpenStack)

Retrieve the Windows instance password (Public Cloud)

  1. Select Project, Instances. In the Actions column for you instance, select Retrieve Windows Password.

  2. In the Retrieve Windows Password page, copy and paste the contents of your private key (pem) file. If you created the key with a password, provide the password as well. Click Retrieve Windows Password.

  3. The Windows password will be displayed.

Retrieve the Windows instance password (HPE Helion OpenStack)

To retrieve the Administrator password for a Windows instance on HPE Helion OpenStack, execute the following:

nova get-password <instance_name> <path_to_private_key>
