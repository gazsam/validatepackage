HPE Helion 2.0 Development Platform: Application Lifecycle Service Documentation

The Application Lifecycle Service (ALS) is a <tm tmtype="tm">Cloud Foundry</tm> -based, managed runtime environment for applications. It is polyglot Platform-as-a-Service (PaaS) software. Whether you run it in your own data center using the hypervisor of your choice or on your favorite cloud hosting provider, it provides the backbone for your private or public PaaS.
