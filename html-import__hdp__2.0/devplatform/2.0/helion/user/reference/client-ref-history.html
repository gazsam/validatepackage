HPE Helion 2.0 Development Platform: ALS Command Reference: History

  • helion history clear: Clear the saved history.
  • helion history limit: Limit the size of the history or display the current limit.
  • helion history list: Show the saved history of commands.

Syntax

helion [options] command [arguments] [command-options]

For more information, use the helion help or helion help [command] commands.

helion history clear

Clear the saved history.

Option
None.

helion history limit <n>

Limit the size of the history. If no limit is specified in the command, the current limit is displayed.

Option
None.

helion history list

Show the saved history of commands.

Option
None.
