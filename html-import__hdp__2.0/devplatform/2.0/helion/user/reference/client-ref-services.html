HPE Helion 2.0 Development Platform: ALS Command Reference: Services

  • helion marketplace: List the supported service plans of the target for the current or specified space.
  • helion purge-service-offerings: Purge all offerings of the service type from the system.
    Warning: Removes ALL offerings. Use with caution.
  • helion service: Show information about the named service.
  • helion services: List the supported and provisioned services of the target.

Syntax

helion [options] command [arguments] [command-options]

For more information, use the helion help or helion help [command] commands.

helion marketplace

List the supported service plans of the target, for the current or specified space.

Option Description
--json Print raw json as output, not human-formatted data.
--organization, -o The once-off organization to use for the current operation.
--space The once-off space to use for the current operation, specified by name. Cannot be used together with --space-guid.
--space-guid The once-off space to use for the current operation, specified by guid. Cannot be used together with --space.
--target The once-off target to use for the current operation.
--token The once-off authentication token to use for the current operation.
--token-file Path to an existing and readable file containing the targets and authorization tokens.

helion purge-service-offerings <service>

Purge all offerings of the service type from the system.
Warning: Do this only for services that have been "orphaned" because their brokers have already been killed.
Option Description
--target The once-off target to use for the current operation.
--token The once-off authentication token to use for the current operation.
--token-file Path to an existing and readable file containing the targets and authorization tokens.
--version The service version. Use this to disambiguate between multiple versions of the same vendor/type.
