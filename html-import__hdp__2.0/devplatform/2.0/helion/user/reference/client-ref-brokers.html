HPE Helion 2.0 Development Platform: ALS Command Reference: Brokers

  • helion add-service-broker: Make the named service broker known.
  • helion create-service-broker: Make the named service broker known.
  • helion delete-service-broker: Remove the named service broker from the target
  • helion remove-service-broker: Remove the named service broker from the target.
  • helion service-brokers: Show the list of known service brokers.
  • helion update-service-broker: Update the target's knowledge of the named service broker.

Syntax

helion [options] command [arguments] [command-options]

For more information, use the helion help or helion help [command] commands.

helion add-service-broker <name>

Make the named service broker known to the system.

Option Description
--organization, -o The once-off organization to use for the current operation.
--space The once-off space to use for the current operation, specified by name. Cannot be used together with --space-guid.
--space-guid The once-off space to use for the current operation, specified by guid. Cannot be used together with --space.
--target The once-off target to use for the current operation.
--token The once-off authentication token to use for the current operation.
--token-file Path to an existing and readable file containing the targets and authorization tokens.
--url Location of the broker.
--username Name of the user to use for access to the broker.
