HPE Helion 2.0 Development Platform: ALS Command Reference: Miscellaneous

  • helion admin: A set of administrative tasks.
  • helion alias: Create a shortcut for a command.
  • helion aliases: List the known aliases (shortcuts).
  • helion color: Management of terminal colors.
  • helion curl: Run a raw REST request against the chosen target.
  • helion drain: Commands for the management of drains attached to applications.
  • helion guid: Map the specified name into a GUID given the type.
  • helion help: Retrieve help for a command or command set.
  • helion named-entities: List the entity types usable for GUID.
  • helion trace: Print the saved REST trace for the last client command to stdout.
  • helion unalias: Remove a shortcut by name.

Syntax

helion [options] command [arguments] [command-options]

For more information, use the helion help or helion help [command] commands.

helion curl <operation> <path> <header>

Run a raw REST request against the chosen target.

Option Description
--data, -d Payload to use for PUT and POST. Cannot be used with neither GET nor DELETE. A value of "-" or "stdin" causes the client to read the data from stdin.
--form Zero or more form fields to be added to the base url. Data is in the form of "key: value".
--output, -o Path to the file to write the returned payload to. A value of "-" or "stdin" causes the client to write the data to stdout.
--port The port to use for the request. Only relevant to websocket operation.
--reconnect Automatically reconnect when the websocket server closes the connection. Only relevant to websocket operation.
--show-extended Show additional information about the request; for example, response headers.
--target The once-off target to use for the current operation.
--token The once-off authentication token to use for the current operation.
--token-file Path to an existing and readable file containing the targets and authorization tokens.
--web-socket, -ws Switch to WebSocket mode for GET. Cannot be used with any other operation.

helion help <CommandName>

Retrieve help for a command or command set. Without an argument, help for all commands is given. The default format is --full.

Option Description
--full Activate full form of the help.
--json Activate json form of the help.
--list Activate list form of the help.
--short Activate short form of the help.
--width, -w The line width to format the help for. Defaults to the terminal width, or 80 when no terminal is available.

helion named-entities

List the entity types usable for 'guid'. I.e. the types of the named entities known to the client.

Option Description

helion trace
