HPE Helion 2.0 Development Platform: Application Lifecycle Service Client Command Reference

Many of the informational commands take a --json option if you wish to generate machine-parseable output. In some cases the json output reveals additional details.

Note that Administrative user privileges are required for some commands.

Getting Started

  • helion login: Logs in to the current or specified target with the named user.
  • helion logout: Logs the current user out of the current, specified, or all targets.
  • helion target: Set the target API endpoint for the client or report the current target.

Administration

  • helion admin grant: Grant the named user administrator privileges for the current or specified target.
  • helion admin list: Show a list of the administrators for the current or specified target.
  • helion admin patch: Apply a patch to the current or specified target.
  • helion admin report: Retrieve a report containing the logs of the current or specified target.
  • helion admin revoke: Revoke administrator privileges for the named user at the current or specified target.
  • helion info: Show the basic system and account information.
  • helion show-token: Print the current target's authorization token.
  • helion stacks: List the supported stacks of the target.
  • helion targets, helion tokens: List the available targets and their authorization tokens, if any.
  • helion usage: Show the current memory allocation and usage of the specified or current space.
  • helion user: Show the name of the current user in the current or specified target.
  • helion user-info: Shows the information of a user in the current or specified target.
  • helion version: Print the version number of the client.

Applications

  • helion app: Show the information of the specified application.
  • helion apps: List the applications deployed to the target.
  • helion list: List the applications deployed to the target.
  • helion releases: Show the list of versions for the specified application.
  • helion versions: Show the list of versions for the specified application.

Authentication Tokens

  • helion create-service-auth-token: Create a new service authentication token.
  • helion delete-service-auth-token: Delete the specified service authentication token
  • helion service-auth-tokens:Show all service authentication tokens known to the target.
  • helion update-service-auth-token: Update the specified service authentication token.

Brokers

  • helion add-service-broker: Make the named service broker known.
  • helion create-service-broker: Make the named service broker known.
  • helion delete-service-broker: Remove the named service broker from the target
  • helion remove-service-broker: Remove the named service broker from the target.
  • helion service-brokers: Show the list of known service brokers.
  • helion update-service-broker: Update the target's knowledge of the named service broker.

Buildpacks

  • helion buildpacks: Show all buildpacks known to the target in the order they are checked in during auto-detection.
  • helion create-buildpack: Add a buildpack to the target.
  • helion delete-buildpack: Remove the specified buildpack from the target.
  • helion rename-buildpack: Change the name of the specified buildpack.
  • helion update-buildpack: Change the information known about the specified buildpack.

Control

  • helion migrate: Move the application to a different space.
  • helion restage: Restage an application and regenerate its droplet.

Domains

  • helion create-domain: Create a new domain.
  • helion delete-domain: Delete the named domain.
  • helion domains: List the available domains in the specified space or all domains.

Feature Flags

  • helion disable-feature-flag: Deactivate the specified feature.
  • helion enable-feature-flag: Activate the specified feature.
  • helion feature-flag: Show the details of the specified feature flag.
  • helion feature-flags: Show all known feature flags and the status of each one.

History

  • helion history clear: Clear the saved history.
  • helion history limit: Limit the size of the history or display the current limit.
  • helion history list: Show the saved history of commands.

Information

  • helion crashes: List recent application crashes.
  • helion crashlogs: Display log information for the application.
  • helion disk: Show the disk reservation for a deployed application.
  • helion drain list: Show the list of drains attached to the application.
  • helion drains: Show the list of drains attached to the application.
  • helion env: List the application's environment variables.
  • helion events: Show recorded application events for an application or space.
  • helion file: Display directory listing or file.
  • helion files: Display directory listing or files.
  • helion health: Report the health of the specified application(s).
  • helion instances: List application instances for a deployed application.
  • helion logs: Display the application log stream.
  • helion mem: Show the memory reservation for a deployed application.
  • helion stats: Display the resource usage for a deployed application.
  • helion tail: Monitor file for changes and stream the changes.

Management

  • helion create-app: Create an empty application with the specified configuration.
  • helion dbshell: Invoke interactive db shell for a bound service.
  • helion delete: Delete the specified application(s).
  • helion drain add: Attach a new named drain to the application.
  • helion drain delete: Remove the named drain from the application.
  • helion env-add: Add the specified environment variable to the named application.
  • helion env-del: Remove the specified environment variable from the named application.
  • helion map: Make the application accessible through the specified URL (a route consisting of host and domain).
  • helion open: Open the url of the specified application in the default web browser.
  • helion push: Configure, create, push, map, and start a new application
  • helion rename: Rename the specified application.
  • helion restart: Stop and restart a deployed application.
  • helion rollback: Switch to a different version of the application.
  • helion run: Run a specified command on a running instance.
  • helion scale: Update the number of instances, memory and/or disk reservation for a deployed application.
  • helion scp: Copy source files and directories to the destination.
  • helion set-env: Add the specified environment variable to the named application.
  • helion ssh: SSH to a running instance (or target), or run a specific command.
  • helion start: Start a deployed application.
  • helion stop: Stop a deployed application.
  • helion switch-version: Switch to a different version of the application.
  • helion unmap: Unregister the application from a URL.
  • helion unset-env: Remove the specified environment variable from the named application.

Miscellaneous

  • helion admin: A set of administrative tasks.
  • helion alias: Create a shortcut for a command.
  • helion aliases: List the known aliases (shortcuts).
  • helion color: Management of terminal colors.
  • helion curl: Run a raw REST request against the chosen target.
  • helion drain: Commands for the management of drains attached to applications.
  • helion guid: Map the specified name into a GUID given the type.
  • helion help: Retrieve help for a command or command set.
  • helion named-entities: List the entity types usable for GUID.
  • helion trace: Print the saved REST trace for the last client command to stdout.
  • helion unalias: Remove a shortcut by name.

Organizations

  • helion create-org: Create a new organization.
  • helion command-delete-org: Delete the named organization
  • helion link-user-org: Add the specified user to the named organization in various roles.
  • helion org: Show the named organization's information.
  • helion orgs: List the available organizations.
  • helion org-users: Show the users for the organization by role.
  • helion quota-org: Set the quotas for the current or named organization.
  • helion rename-org: Rename the named organization.
  • helion switch-org: Switch the current organization to the named organization.
  • helion unlink-user-org: Remove the specified user from the named organization in various roles.
  • helion update-org: Change one or more attributes of an organization in a single call.

Placement Zones

  • helion placement-zone: Show the list of DEAs associated with the specified placement zone.
  • helion placement-zones: Show the available placement zones.
  • helion set-placement-zone: Associate the application with a specific placement zone.
  • helion unset-placement-zone: Remove the association between an application and its current placement zone.

Quotas

  • helion quota configure: Reconfigure the named quota plan.
  • helion quota create: Create a new quota plan.
  • helion quota delete: Delete the named quota plan.
  • helion quota list: List the available quota plans.
  • helion quota rename: Rename the named quota plan.
  • helion quota show: Show the details of the named quota plan
  • helion quotas: List the available quota plans.

Routes

  • helion delete-route: Delete the named route.
  • helion routes: List all routes or routes available in the specified space.

Security Groups

  • helion bind-security-group: Bind the named security group to the current space, the specified space, or the running and staging phases.
  • helion create-security-group: Create a new security group.
  • helion delete-security-group: Delete the named security group.
  • helion security-group: Show the details of the named security group.
  • helion security-groups: Show all known security groups or just those assigned to the running and staging phases.
  • helion unbind-security-group: Drop the named security group from the current space, the specified space, or the running and staging phases.
  • helion update-security-group: Modify the named security group.

Service Management

  • helion bind-service: Bind the named service to the specified application.
  • helion clone-services: Copy the service bindings of the source application to the destination application.
  • helion create-service: Create a new provisioned service and optionally bind it to an application.
  • helion delete-service: Delete the named provisioned service.
  • helion rename-service: Rename the specified service instance.
  • helion tunnel: Create a local tunnel to a service and optionally start a local client as well.
  • helion unbind-service: Disconnect the named service from the specified application.
  • helion update-user-provided-service: Update the credentials of a user provided service.

Service Plans

  • helion hide-service-plan: Hide the specified service-plan from the named organization.
  • helion service-plan: Show the details of the specified service plan
  • helion service-plans: List all available plans of the supported services
  • helion show-service-plan: Make the specified service plan visible to the named organization.
  • helion update-service-plan: Update the target's knowledge of the named service plan.

Services

  • helion marketplace: List the supported service plans of the target for the current or specified space.
  • helion purge-service-offerings: Purge all offerings of the service type from the system.
    Warning: Purges ALL offerings. Use with caution.
  • helion service: Show information about the named service.
  • helion services: List the supported and provisioned services of the target.

Spaces

  • helion create-space: Create a new space.
  • helion delete-space: Delete the named space.
  • helion link-user-space: Add the specified user to the named space in various roles.
  • helion rename-space: Rename the named space.
  • helion space-users: Show the users for the space, by role.
  • helion space: Show the named space's information.
  • helion spaces: List the available spaces in the specified organization.
  • helion switch-space: Switch from the current space to the named space.
  • helion unlink-user-space: Remove the specified user from the named space in various roles.
  • helion update-space : Change one or more attributes of a space in a single call.

Space Quotas

  • helion create-space-quota: Create a new space quota.
  • helion delete-space-quota: Delete the named space quota.
  • helion rename-space-quota: Rename the named space quota.
  • helion set-space-quota: Assign the specified space quota to the current or specified space
  • helion space-quota: Show the details of the named space quota.
  • helion space-quotas: List the space quotas owned by the current, specified, or all organizations.
  • helion unset-space-quota: Drop the space quota from the current or specified space.
  • helion update-space-quota: Modify the named space quota.

User Management

  • helion add-user: Register a new user in the current or specified target. Requires admin privileges.
  • helion delete-user: Delete the named user and the user's applications and services from the current or specified target. Requires admin privileges.
  • helion login-fields: Show the names of the credential fields needed for a login.
  • helion passwd: Change the password of the current user in the current or specified target.
  • helion register: Register a new user in the current or specified target. Requires admin privileges.
  • helion token: Interactively set authentication token.
  • helion unregister: Delete the named user and the user's applications and services from the current or specified target. Requires admin privileges.
  • helion users: Show the list of users known to the current or specified target.
