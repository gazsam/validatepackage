HPE Helion 2.0 Development Platform: ALS Command Reference: Buildpacks

  • helion buildpacks: Show all buildpacks known to the target in the order they are checked in during auto-detection.
  • helion create-buildpack: Add a buildpack to the target.
  • helion delete-buildpack: Remove the specified buildpack from the target.
  • helion rename-buildpack: Change the name of the specified buildpack.
  • helion update-buildpack: Change the information known about the specified buildpack.

Syntax

helion [options] command [arguments] [command-options]

For more information, use the helion help or helion help [command] commands.

helion buildpacks

Show all buildpacks known to the target in the order they are checked in during auto-detection.

Option Description
--json Print raw json as output, not human-formatted data.
--target The once-off target to use for the current operation.
--token The once-off authentication token to use for the current operation.
--token-file Path to an existing and readable file containing the targets and authorization tokens.

helion create-buildpack <name> <zip>

Add a buildpack to the target.

Option Description
--disable Disable the buildpack, preventing its use.
--enable Enable use of the buildpack.
--lock Lock the buildpack against modification.
--position, -p Location of the buildpack in the sequence used to check them during auto-detection.
--target The once-off target to use for the current operation.
--token The once-off authentication token to use for the current operation.
--token-file Path to an existing and readable file containing the targets and authorization tokens.
--unlock Unlock the buildpack, allow changes again.

helion delete-buildpack <name>

Remove the specified buildpack from the target.

Option Description
--target The once-off target to use for the current operation.
--token The once-off authentication token to use for the current operation.
--token-file Path to an existing and readable file containing the targets and authorization tokens.

helion rename-buildpack <name> <newname>

Change the name of the specified buildpack.

Option Description
--target The once-off target to use for the current operation.
--token The once-off authentication token to use for the current operation.
--token-file Path to an existing and readable file containing the targets and authorization tokens.

helion update-buildpack <name>

Change the information known about the specified buildpack.

Option Description
--disable Disable the buildpack, preventing its use.
--enable Enable use of the buildpack.
--lock Lock the buildpack against modification.
--position, -p Location of the buildpack in the sequence used to check them during auto-detection.
--target The once-off target to use for the current operation.
--token The once-off authentication token to use for the current operation.
--token-file Path to an existing and readable file containing the targets and authorization tokens.
--unlock Unlock the buildpack, allow changes again.
--zip Path or url of the new zip file containing the implementation of the buildpack. Accepts the path to a local directory as well, which will become the zip file to upload.
