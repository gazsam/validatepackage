HPE Helion 2.0 Development Platform: Application Lifecycle Service User Guide

The Application Lifecycle Service is a polyglot Platform-as-a-Service (PaaS). You can easily deploy applications written in a wide range of languages and web frameworks using a variety of data services simply by pushing your source code to the system.

Application Lifecycle Service handles the automatic configuration of the language runtime, web server, application dependencies, databases, and other services.

Note: "Worker Applications" are non-HTTP apps, such as background scripts, which run as ALS applications. To deploy worker applications, you need to use the commandkey and set the proceses: web key to Null ("~").
