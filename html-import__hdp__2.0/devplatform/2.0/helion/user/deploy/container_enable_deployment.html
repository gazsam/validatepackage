HPE Helion 2.0 Development Platform: Enable Container Deployment

For security, Docker container deployments are disabled. You can enable container deployment in two ways:
  • Add sudo permissions to users within an organization
  • Disable sudo requirement for Docker apps

To Add sudo Permission for All Users

  1. Open the Helion Management Console.
  2. Click Admin > Quotas.
  3. Click a quota plan and then click Edit Quota.
  4. Check the Allow Sudo checkbox and then click Update.

  5. For the setting to take effect, restart any apps running under the quota plan.

To Remove sudo Requirement for Docker App Deployment

  1. Open the Helion Management Console.
  2. Click Settings > HPE Helion Development Platform Settings.
  3. On the left panel, lick Docker Apps Settings.
  4. Uncheck the Require Sudo checkbox.
