HPE Helion 2.0 Development Platform: Where to Find HipChat Notification Data

Note: You must be the owner of the HipChat room where the notifications will appear.

The Room Number

  1. In the HipChat client, enter the HipChat room where the notifications should appear.
  2. From the Settings dropdown, select Integration. (Or press CTRL-G)
  3. Log in to the HipChat web administration interface.
  4. Where the UI says Rooms / Name of Room click Name of Room to show the room's details.
  5. The API ID number is the one you want.
    Tip: For faster future reference, look at the browser's address bar. The room number also appears at the end of the URL. https://hipchat.hpcloud.net/rooms/show/###

Create a New Token

  1. In the HipChat client, enter the HipChat room where the notifications should appear.
  2. From the Settings dropdown, select integration. (Or press CTRL-G)
  3. Log in to the HipChat web administration interface.
  4. Where the UI says Rooms / Name of Room click Name of Room to show the room's details.
  5. In the menu on the left, click Tokens.
  6. Enter a descriptive name in the Label field and then click Create.
  7. The token will be generated. Copy and paste the entire token from HipChat into the Token field.
