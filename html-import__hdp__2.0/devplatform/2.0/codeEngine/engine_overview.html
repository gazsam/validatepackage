HPE Helion 2.0 Development Platform: Continuous Integration and Deployment Overview

HPE Helion Code Engine is a continuous integration and delivery system designed for small development teams who are interested in building "greenfield" cloud native apps and deploying them with Helion Development Platform.

HPE Helion Code Engine leverages Git version control technologies to automate build, test and deploy cycles so that developers can quickly bring their ideas to life and get customer validation.

More specifically, unit tests are automated through the most popular testing frameworks for the supported tech stacks and apps can be deployed to any specified environment - testing, staging or production.

Continuous integration and development automates processes that traditionally curb agile development in the enterprise. HPE Helion Code engine can catalyse your journey to modern development workflows.

Highlights

  • Automates testing through integration with popular testing frameworks.
  • Supports Git version control technologies: Github.com, Github Enterprise, and on-premise Git repositories.
  • Deploy to any specified Helion Development Platform environment: test, staging or production.
  • Supports applications built on Node.js, Java, and Python.
  • Sends notifications to popular collaboration software such as HipChat and Flowdock.
  • Integrates with StormRunner for post-deployment load-testing analysis.
