HPE Helion 2.0 Development Platform: Resources for Application Developers

HPE has created a series of small, simplified sample applications to help you understand and exercise the Development Platform services.

Each sample builds on the previous one and all source code is provided for you to build, deploy, and take apart. Each sample is provided in multiple languages.

Prerequisites - Before you begin

In order to download and deploy these sample applications, you must have the necessary framework installed and configured.

  • HPE Helion OpenStack® cloud deployed.
  • Application Lifecycle Service Cluster available with MySQL and the Message Broker service enabled.
  • Log on and password credentials for the cluster.
  • Access to the Helion command-line interface (CLI)
  • (Optional) Access to the Eclipse deployment plugin for JAVA applications.
