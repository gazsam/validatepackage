HPE Helion 2.0 Development Platform: List Available HDN Content

There are two ways to retrieve the list the HDN content available to you.

Option 1: Quick list of recent versions only with no installation state

This option lists the most recent versions that are available to you. It does not list every package you could install, and it does not indicate which packages are currently installed.

This is the fastest option if you are just looking for a wscatalog number.

openstack hdpi list --os-cloud mycloud
The list will look similar to the following:
+----------------------------+----------------+-----------+-----------+
| service                    | id             | version   | state     |
+----------------------------+----------------+-----------+-----------+
| Helion Code Engine         | wscatalog.2776 | 1.0.0     | available |
| Helion Code Engine 2.0     | wscatalog.2812 | 1.0.0.42  | available |
| als                        | wscatalog.2765 | 2.0.0.539 | available |
| als                        | wscatalog.2800 | 2.0.0.548 | available |
| database                   | wscatalog.2762 | 2.0.0.32  | available |
| database                   | wscatalog.2803 | 2.0.0.37  | available |
| database                   | wscatalog.2804 | 2.0.0.38  | available |
| messaging                  | wscatalog.2761 | 1.0.0     | available |
| messaging                  | wscatalog.2796 | 1.0.0.165 | available |
+----------------------------+----------------+-----------+-----------+

Option 2: Full list of all versions and installation state

This options lists everything that is available to you. It lists every package you could install, and it also indicates which options are currently installed.

This is a quick way to check whether a service has already been installed.

openstack hdpi list --all --os-cloud mycloud
The list will look similar to the following:
