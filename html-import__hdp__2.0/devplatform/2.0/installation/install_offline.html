HPE Helion 2.0 Development Platform: Downloading for Offline Installation

If downloading directly to the Deployer is not an option for security or other reasons, you can pre-download the installation packages for use during installation.

The pre-download also eliminates the possibility of connection interruptions during installation.

  1. Using your preferred Web browser, navigate to the HPE Helion Download Network (HDN) and log in. If necessary, create an account first.
  2. Click on the Download button in the left navigation panel.
  3. Click the service you wish to download. Click the blue Download button at the top right of the screen to begin the download.
  4. The best option is to download directly to the HPE Helion OpenStack Deployer. If you can not download directly to the HPE Helion OpenStack deployer, copy all the downloaded packages to a USB stick or external hard drive and then copy the files onto the physical machine hosting the HPE Helion OpenStack Deployer. Make sure there is enough space; the file sizes can be significant.
  5. Follow the installation instructions provided for each service.
    Note: When installing pre-downloaded files, make sure you run the installer from the same directory where the installation files are. If the installer cannot find a file with the name you provided in the local directory, it will attempt to download the files again.
